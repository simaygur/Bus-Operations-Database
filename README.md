# Bus Operations Database

Bu repository, otobüs firması yönetim sistemi için oluşturulmuş veritabanı yapısını içermektedir. Veritabanı; bilet satış, sefer yönetimi, koltuk rezervasyonu, şoför yönetimi ve takometre kayıtlarını destekleyecek şekilde tasarlanmıştır.

---

## İçerik

Veritabanı aşağıdaki modülleri desteklemektedir:

- Kullanıcı ve giriş işlemleri
- Sefer ve rota yönetimi
- Bilet ve koltuk rezervasyon sistemi
- Admin yönetim yapısı
- Şoför kayıt ve yönetimi
- Takometre (mesafe / süre / performans) kayıt sistemi

---

## Not

Bu repository yalnızca veritabanı yapısını içermektedir. Uygulama kodları ayrı bir projede yer almaktadır.

---

## Kullanılan Teknoloji

- SQL Server
